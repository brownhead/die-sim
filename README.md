# Die Simulator

This is a simple program for letting you roll a variety of dice.

![Animation of program in use](https://raw.github.com/itsjohncs/die-sim/master/demo.gif)

## How to Run

```shell
git clone https://github.com/itsjohncs/die-sim.git
./die-sim/die.py
```
